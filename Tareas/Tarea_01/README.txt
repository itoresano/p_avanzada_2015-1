Archivos:

En la entrega de la tarea 01, el archivo a ejecutar dentro de python es "main.py" el cual
abre la consola y permite realizar las consultas y calculos pedidos en el enunciado.

---------------------------------------------------------------------------------------------

Funcionalidades del Programa:

Al ejecutar el archivo "main.py" el programa automaticamente carga la "base de datos" 
entregada para la tarea, luego inicia la consola del programa.

Al abrir la consola se muetra un menú de 5 opciones, ennumeradas del 1 al 5, las cuales
realizan las consultas pedidas en esta tarea.

Cada opcion lleva a distintas funciones dentro del programa e interactua con el usuario para
recibir mas datos y poder completar las consultas recibidas.

---------------------------------------------------------------------------------------------

Clases:

Cada clase del programa esta construida en un archivo diferente, los cuales tienen el nombre
de la clase, cada clase tiene su inicializador y aparte un metodo __str__ para poder imprimir
los atributos.
La clase Vehiculo tiene sub clases (transporte pasajeros y transporte carga) las cuales a su
vez tienen clases hijas ( Avion, Bus etc...)
